# Superlines AI Search Intelligence

Cursor plugin for AI Search visibility tracking, Generative Engine Optimization (GEO), and content optimization across 10+ AI platforms.

## What's included

### MCP Server Integration

Connect directly to the Superlines analytics engine with 26+ tools:

- **Brand analytics** — visibility, citations, share of voice, mentions, sentiment
- **Competitive intelligence** — gap analysis, competitor insights, fan-out query analysis
- **Webpage auditing** — technical SEO, content quality, Schema.org optimization
- **Strategic planning** — action plans, content opportunities, best performing prompts
- **Prompt management** — add, pause, unpause, and label tracking prompts

## Setup

### 1. Get your API key

Sign up at [superlines.io](https://superlines.io) and generate an API key from your dashboard settings.

### 2. Set environment variable

Add your Superlines API key as an environment variable:

```
SUPERLINES_API_KEY=your-api-key-here
```

### 3. Install the plugin

Install from the Cursor Marketplace or add the plugin files to your project.
